# Starter Project

My starter project is a customizable Arduino project. I soldered a force sensor onto a proto shield and connected a servo motor to it. The code I wrote controlled the angle of the servo motor based on the pressure from the force sensor. I enjoyed soldering the wires to the proto sheild.
